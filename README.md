# Mini Project 3 — Balance API

Mini Project 3 is a backend API built with **Node.js**, **Express**, **TypeScript**, **MongoDB**, **Mongoose**, and **dotenv**.

The project allows creating a user with an initial balance, current balance, and an empty transactions history.

---

## Tech Stack

- Node.js
- Express
- TypeScript
- MongoDB
- Mongoose
- dotenv
